# Análisis de Exoplanetas

Este repositorio contiene un proyecto para el **análisis y estudio de exoplanetas**, basado en un conjunto de datos y visualizaciones mediante un **cuaderno Jupyter Notebook** y un informe en PDF.

## Contenido

- **`Garcia_Nicolas_exoplanetas.ipynb`**  
  Notebook principal con código para analizar datos de exoplanetas, generar gráficos y extraer conclusiones sobre sus características.

- **`Garcia_Nicolas_exoplanetas.pdf`**  
  Documento en PDF que complementa el análisis con explicaciones, resultados y conclusiones interpretadas.

## ¿Qué es un exoplaneta?

Un **exoplaneta** es un planeta que orbita una estrella diferente a nuestro Sol. En las últimas décadas se han descubierto miles de exoplanetas, lo que ha ampliado enormemente nuestro conocimiento sobre sistemas planetarios fuera del sistema solar. :contentReference[oaicite:1]{index=1}

## Requisitos

Para ejecutar el notebook se recomienda tener:

- Python 3.x
